## Enfasis de Texto

* Para nosotros utilizar un texto en **negrita** se utilizan dos "**" (asterisco).
* Para nosotros utilizar un texto en *cursiva* vamos a utilizar un "*" (asterisco).
* Para nosotros ~~tachar un texto~~ necesitas usar "~".
