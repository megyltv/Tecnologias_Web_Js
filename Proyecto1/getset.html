Getter and Setter in Classes

metodos de propiedades

Descripcion

ES6 ha empezado a soportar las funciones getter y setter dentro de la definicion de clases.

Caracteristicas

  • 1. La funcion getter permite traer datos, generalmente se utiliza en una clase para traer datos que se encuentran definidos.
  • 2. La funcion setter permite setear datos en las variables de un objeto. Esta funcion, tambien puede ser usada en un objeto para reasignar nuevos valores al objeto.
  • 3. Para la declaracion de estas funciones, no es necesario utilizar la palabra function.

Ejemplo descriptivo

                    
var libro = {
    titulo: 'Divergente',
    autor: 'VeronicaRoth',
    //Se inicializa el objeto con valores los cuales se
    //obtienen con la funcion get 
    get infoLibro() {
        console.log('Informacion del Libro');
        return this.titulo + ' ' + this.autor;
    },
    //Funcion set recibe un objeto con valores
    set infoLibro(libro) {
        console.log('Seteo de Valores nuevos');
        //Asigno nuevo valores
        var palabras = libro.toString().split(' ');
        this.titulo = palabras[0] || '';
        this.autor = palabras[1] || '';
    }
}

person.infoLibro; // Divergente VeronicaRoth
person.infoLibro = 'Oscuros LaurenKate';
person.infoLibro; // Oscuros LaurenKate
