Classes

¿orientado a objetos?

Descripcion

En ES6 se ingresa la definicion de clases, las cuales estan compuestas de atributos y metodos o propiedades.

Caracteristicas

  • 1. Una clase esta conformada por el metodo constructor.
  • 2. Una clase puede contener atributos y metodos propios.
  • 3. Ya no es necesario el uso de prototipos para alterar el valor de un objeto, que antes se definia como funcion.
  • 4. Una clase puede heredar de otra utilizando la palabra extend.

Ejemplo descriptivo

                    
//Antes
//Se define objeto como funcion
function Persona(nombre, edad, telefono) {
    this.nombre   = nombre;
    this.edad    = edad;
    this.telefono = telefono;
}

//Se accede a los datos con prototype y se define una funcion
//para hacer algun calculo.

Persona.prototype.datosPersonales = function () {
    return return nombre+' '+edad;
};
               
                                             
//Con ES6
//Se define como clase
class Persona {
   //Se utiliza para instanciar un objeto de clase Persona
    constructor(nombre, edad, telefono) {
        this.nombre   = nombre;
        this.edad    = edad;
        this.telefono = telefono;
    }

   //Se definen operaciones extras sobre el objeto con metodos
    datosPersonales() {
      return nombre+' '+edad; 
    }
}
