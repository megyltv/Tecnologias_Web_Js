Arrow Functions

Ejemplo descriptivo

Descripcion

La expresión de Arrow function o función flecha (también conocida como función flecha gruesa) dispone de una sintaxis más corta comparada con la expresión de función convencional y vincula contextualmente el valor de this.

Caracteristicas

  • 1. Proveen una sintaxis reducida y this contextual
  • 2. Se encuentra formado por dos partes: Identificador => Expresion
  • 3. Se evita o se salta escribir function y return, y en ciertos casos parentesis, corchetes o llaves
  • 4. A diferencia de las funciones function, las Arrow Function no tienen su propio this, sino que su valor es siempre heredado del alcance en el que se encuentre
  • 5. El return se encuentra en la expresion de forma explicita.
                    
//Se especifica la function y return
var suma1 = (function (num1, num2) {
    return num1 + num2;
})
console.log(suma1(5, 7)); //12

//Arroy Function
//Identificador => Expresion
var suma2 = ((num1, num2) => num1 + num2);
console.log(suma2(5, 7)); //12

        
var arregloNum = [4, 10, 5, 6];

// Con Function
var arregloNum2 = arregloNum.map(function (v) {
    return v=v*2; 
}); // [8,20,10,12]
console.log(arregloNum2);

//Con Arrow Function
var arregloNum3 = arregloNum.map(v => v * 2); // [8,20,10,12]
console.log(arregloNum3);
