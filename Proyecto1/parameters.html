Default Parameters, Rest Parameters, Named Parameters

parametros

Descripcion

Existen tres tipos de parametros que pueden ser utilizados dentro de las funciones> default values, rest parameters y named parameters.

Caracteristicas

  • 1. En las Default Paremeters se puede definir valores por defecto que pueden tomar las variables en caso de que algun argumento falte al momento de llamar
  • 2. Rest Parameters permite enviar como argumento un numero indefinido de parametros.
  • 3. Named Parameters, en ES5 puede ser enviando un objeto o utilizando "destructuring", y enviando cada variable especifica con valores

Ejemplo descriptivo

Default Parameters

                    
function multiplicacion(num1, num2) {
    num1 = num1 || 0; // || valor por default
    num2 = num2 || 0; // || valor por default
    return num1 * num2;
}
               	
multiplicacion(5,7); // 35, num1=5 y num2=7
multiplicacion(5); // 0, num1=5 y num2=0 
multiplicacion(); // 0, num1=0 y num2=0

Rest Parameters

                    
//Envio de parametros
function listaNumeros(num, ...params) {
  //Recorro params
   for(let param of params){
   		num=num+param;
   }
    return num};
}
result = listaNumeros(1, 2, 3); // num=1, params=[2,3]

Named Parameters

                    
function valores({nombre='Dalia', apellido='Perez', edad=10}={}){
    //Las variables son asignadas con valores y pueden ser utilizados
    //dentro de la funcion
    
	return nombre+apellido+edad;               	
}
console.log(valores()); // DaliaPerez10
