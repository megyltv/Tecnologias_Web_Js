Destructuring arrays

arreglos

Descripcion

Destructuring Objects o Destruyendo de Objetos permite manipular los objetos de una forma mas sencilla.

Caracteristicas

  • 1. Permite extraer valores de los objetos de una forma sencilla
  • 2. Permite extraer y almacenar el valor de un componente del objeto en otra variable.
  • 3. Permite leer una estructura en una sentencia
  • 4. Permite realizar una asginacion sin declarar la sentencia para asignar
  • 5. Permite intercambiar valores entre variables sin usar auxiliares.

Ejemplo descriptivo

                    
//Sin destructuring
var hermana = {
    ocupacion: 'estudiante',
    edad: '15',
    mascotas:2
};
//Se realiza tres asignaciones
var ocupacion = hermana.ocupacion; // 'estudiante'
var edad = hermana.edad; // '15'
var mascotasNum = hermana.mascotas; // 2
        
//Con destructuring
let hermana2 = {
    ocupacion: 'estudiante',
    edad: '15',
    mascotas:2
};
        
//Asignacion de valores en una sola linea
let { ocupacion, edad, mascotas } = hermana2;

console.log(ocupacion); // 'estudiante'
console.log(edad); // '15'
console.log(mascotas); // 2
