Spread Operator

operadores

Descripcion

El operador Spread permite manipular los valores de arreglos, pues permite copiar los valores a la expresion que se requiera.

Caracteristicas

  • 1. El operador Spread se expresa (...)
  • 2. Este operador permite copiar propiedades de un objeto a otro de una forma sencilla
  • 3. Lee un arreglo como si se fuera similar a una expresion para hacer iteraciones

Ejemplo descriptivo

                    
                    
var numeros, cadena, mezcla, result1, result2;
numeros = [1, 2, 3];
cadena = "hola";
mezcla = [42, "mundo"];

// UUtilizando un metodo de concatenacion.
result1 = numeros.concat(cadena, mezcla);

// Usando Spread (...) como operador para concatenar
result2 = [...numeros, cadena, ...mezcla];
