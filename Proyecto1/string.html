Strings .includes() .repeat()

operaciones strings

Descripcion

Las opciones de la libreria estandar de ES6 ha incrementado y, dentro de esos cambios se encuentran los nuevos metodos de string, como .include() y .repeat().

Peculiaridades

  • 1. La libreria .include() permite comprobar que una cadena dentra elementos.
  • 2. .include() retorna un bool como valor de retorno.
  • 3. El metodo .include() permite verificar que una cadena esta contenida en otra.
  • 4. Permite concatenar un numero indicado de veces, la misma cadena.
  • 5. Estos metodos son sensibles al teclado

Ejemplo descriptivo

.include()

                    
var str = 'Hola mundo, frase ser o no ser, esa es la pregunta.';

//.include(Substring a buscar)
console.log(str.includes('Hola')); // true - consta en la cadena
console.log(str.includes('es la pregunta')); // true - consta en la cadena
console.log(str.includes('hola Mundo')); // false - Compara mayusculas y minusculas
console.log(str.includes('SER', 1)); // false - Compara mayusculas y minusculas
console.log(str.includes('Pregunta')); // false - Compara mayusculas y minusculas

.repeat()

                    
//Sin el metodo de repeat, se concatena con un a funcion de lazo
function repeat(string, numero) {
   var strings = [];
   while (strings.length 'menor' numero) {
         strings.push(string);
   }
   return strings.join('');
}
        
console.log(repeat('mundo',3));

// String.repeat(numero de repeticiones)
//valorString.repeat();
console.log('hola'.repeat(3)); // holaholahola
        
//Se puede definir una variable y a partir de ella llamar a la funcion
var cadena='mundo';
console.log(cadena.repeat(3)); // mundomundomundo
