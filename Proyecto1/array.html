Destructuring arrays

arreglos

Descripcion

Destructuring o Destruyendo Arreglos permite manipular los arreglos de una forma mas sencilla.

Caracteristicas

  • Permite extraer valores de los arreglos de una forma sencilla
  • Permite extraer y almacenar el valor de un componente del arreglo en otra variable.
  • Permite leer una estructura en una sentencia
  • Permite realizar una asginacion sin declara la sentencia para asignar
  • Permite intercambiar valores entre variables sin usar auxiliares.

Ejemplo descriptivo

                    
var arreglo = ["uno", "dos", "tres"];

// sin destructuring, se asigna valores a cada una
var uno = arreglo[0]; //"uno"
var dos = arreglo[1]; //"dos"
var tres = arreglo[2]; //"tres"

// con destructuring
var [uno, dos, tres] = arreglo; // asignación en una sola linea

var num1, num2;

[num1, num2] = [1, 2];


var nombre = Maria;
var apellido = Sanchez;

[nombre, apellido] = [apellido, nombre]; //[Sanchez, Maria]
