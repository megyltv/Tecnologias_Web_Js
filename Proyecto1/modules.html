Modules export and import

codigo mantenible

Descripcion

ES6 provee la funcionalidad en Javascript de construir modulos, los cuales contienen un porcion de codigo a ser ejecutada. Estos pueden ser importados o exportados de acuerdo a lo que se necesite en el codigo.

Caracteristicas

  • 1. Cada modulo es un pedazo de codigo que es ejecutado una vez este cargado
  • 2. Un modulo puede contener declaraciones de variables, funciones, entre otros.
  • 3. Un modulo puede importar cosas de otros modulos, mediante el uso de paths.
  • 4. Los modulos a pesar de ser importados varias veces, unicamente presentan una instancia.
  • Descripcion 5

Ejemplo descriptivo

Exportar Modulos

                    
//Exportar modulos

//Exportar en archivos funciones de librerias propias.
export const sqrt = Math.sqrt;

//Exportar funciones con operaciones que se han definido por el usuario.
export function square(x) {
    return x * x;
}

//Exportar una funcion con sus operaciones
export function diag(x, y) {
    return sqrt(square(x) + square(y));
}

//Exportar Valores
export let name = 'Hola Mundo';

Importar Modulos

                    
//Importar modulos

//Importar un archivo completo
import 'hola'; //Se ejecuta todo el codigo que tenga.

//Importar mas de un archivo y se indica la ubicacion o path.
import { square, diag }	from 'lib'; 

//Se realizan operaciones sobre lo importado. De acuerdo al modulo se envian
//parametros
console.log(square(11)); // 121 (Se ejecuta como funcion)
console.log(diag(4, 3)); // 5
