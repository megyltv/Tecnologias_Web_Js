var vs let vs const

Declaracion de Variables

Descripcion

Las opciones de var, let y const estan disponibles en el Ecmascript 2015 y son diferentes formas que permiten la declaracion de variables para el almacenamiento de valores.

Caracteristicas

  • 1. A diferencia de var, la declaraciones con let y const no son llevadas al inicio del alcance en el cual estan siendo utilizadas (no hoisted)
  • 2. let y const tienen un alcance limitado, por lo cual si son referenciados antes de ser declarados producira un error.
  • 3. Const se utiliza, generalmente, para variables cuyo valor permacera constante
  • 4. Let se utiliza para variables cuyo valor variara en el tiempo
  • 5. Let se utiliza generalmente dentro de funciones, en donde se necesita definir variables temporales.

Ejemplo descriptivo

Var y Let

                    
// var es usado en el JavaScript antiguo.
var a = 12;

// let es usado con el nuevo Ecmascript 2015
var i = "i";
var temporal = "temporal";

for (let i = 1; i != false; i = false) {
    // Se crean temporalmente las variables i y temporal, dentro del alcance del for.
    let temporal = 10;
    console.log(i, temporal); // Se modifican valores dentro del for = 1 y 10
}
console.log(i, temporal); // Variables toman el valor definido afuera: i y temporal


for (var i = 1; i != false; i = false) {
    var temporal = 10;
}
console.log(i, temporal); // Se modifican los variables de forma global: false 10

Const

                    
// const es usado para declarar objetos que no se van a modificar, pero si se pueden aumentar

const a = 20;
console.log(a); //20

a=(a+2)*5+(a*10); // Error, por reasignar valor a una constante---> No cambia el valor de a.
console.log(a); 


// En arreglos, si es constante si puede aumentar y disminuir valores

const arrayGrande = ['a','b','c'];
arrayGrande.push('d'); // NO hay error const arrayGrande = ["a", "b", "c", "d"]
console.log(arrayGrande);
arrayGrande.pop(); // ["a", "b", "c"]
console.log(arrayGrande);
arrayGrande.pop(); // ["a", "b"]
console.log(arrayGrande);
