IIFEs ahora Blocks

problemas en bloques

Descripcion

IIEF o Immediately Invoked Function Expressions permite encerar los valores para que sean usados dentro de su alcance. En ES6, ahora se tiene la opcion de crear bloques o funciones.

Caracteristicas

  • 1. Permite la creacion de alcances basados en bloques, siendo una alternativa al uso de funciones.
  • 2. Los bloques son como funciones, puesto que en su interior pueden realizar operaciones sobre variables.
  • 3. Los bloques permiten encerrar el alcance de una variable dentro de los parentesis
  • 4. Antes, con IIEF, para encerrar una variable era necesario declarar a la funcion como se encuentra en el Ejemplo. Con los bloques en cambio, se indica el inicio y fin, y las variables se indican de tipo let.

Ejemplo descriptivo

                    
// Funciones IIEF para encerrar a una variable var
(function () {
    var fruta = 'Manzana';
    console.log(fruta); // Reference Error
}());
console.log(fruta); // Reference Error
        
//Bloques
{
    //Permite realizar operaciones, dentro del alcance definido entre {}
    let fruta = 'Manzana';
    console.log(fruta);
};
